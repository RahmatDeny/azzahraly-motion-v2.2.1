# Azzahraly Motion v2.2.1

Proyek web berbasis **React + Vite** untuk Azzahraly Motion.

---

## Cara Menjalankan Proyek

### 1. Clone Repository

```bash
git clone https://github.com/RahmatDeny/azzahraly-motion-v2.2.1.git
```

### 2. Masuk ke Folder Proyek

```bash
cd azzahraly-motion-v2.2.1
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Jalankan Development Server

```bash
npm run dev
```

Setelah berhasil, buka browser dan akses:

```
http://localhost:5173
```

- [React](https://react.dev/)
- [Vite](https://vitejs.dev/)
